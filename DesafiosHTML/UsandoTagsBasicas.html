Tags Usadas em HTML


Dentro de uma pagina podemos ter vários tipos de tags, como por exemplo:

  • H1, H2,H3,H4,H5,H6
  • small
  • mark
  • i
  • u
  • strong
  • ol
  • ul
  • li
  • a
  • hr
  • sub
  • blockquote

Então agora vamos explicar cada uma das tags citadas acima

  1. H1 até H6:

    2. Essa tags são usadas para determinar o grau de importância de cada titulo usado dentro da página


  2. small:

    3. é utilizado para reduzir o tamanho da fonte num intervalo definido de um texto. exemplo: Texto dentro da tag small


  3. mark:

    4. destaca um trecho do conteúdo do documento. exemplo: Texto dentro da tag mark


  4. i:

    5. pode representar uma parte do texto com uma diferença semântica, na qual a representação tipográfica padrão é no tipo itálico exemplo: Texto dentro da tag i


  5. u:

    6. é utilizado para sublinhar um espaço definido de um texto exemplo: Texto dentro da tag u


  6. strong:

    7. é utilizado em conteúdos que são de "grande importância", incluindo coisas de urgentes (como alertas). exemplo: Texto dentro da tag u


  7. ol, ul e li

    8. ol seria para listas ordenas.ul seria para listas não ordenadas e li seria os itens dessa listas.


  8. a:

    9. é usada para criar uma âncora de onde vincular. Exemplo de texto com link usando a tag a: Texto com Link


  9. hr:

    10. É um elemento HTML que representa uma quebra de linha temática entre elementos HTML.
    É exibido como uma linha horizontal separando os conteúdos de uma página web. Exemplo usando a tag hr:


  10. sub:

    11. é utilizado escrever um texto com letras menores e com o alinhamento abaixo das que do resto do conteúdo. Exemplo: usando a tag sub


  11. blockquote:

    12. Indica que o texto incluído é uma longa citação. Normalmente, este é processado visualmente pelo recuo Exemplo:

    usando a tag blockquote

Tags Extras

  • font: é utilizado para configurar o tipo, cor e tamanho das letras de um texto de uma página HTML.

  • p: usada para criar paragrafos e já usada nas listas acima.

  • del: é usada para identificar o texto que foi excluído de um documento, mas retido para mostrar o histórico de modificações feitas no documento
    Exemplo: Usando a tag del

  • abbr: Abreviatura é o marcador utilizado para se inserir uma sigla em uma página HTML.
    Exemplo: Posicione o cursor do mouse sobre a sigla "IFTM": IFTM
